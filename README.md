# sankey-finanzfluss-diagramm
creats sankey diagramm from csv file

Die csv datei muss wie folgt aufgebaut sein:
Datum,Kategorie,Einkommen,Ausgaben,Zweck

z.B. <br>
1/1/2013,Gehalt,306.65,,ZWECK: EINNAHME <br>
1/1/2013,Fixkosten,,100.43,ZWECK (AUSGABE)

oder<br>
xlsx Datei von [Expense Manager](https://play.google.com/store/apps/details?id=at.markushi.expensemanager&hl=gsw) by Markus Hintersteiner
